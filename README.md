# VOLT Recovery - Premium Landing Page ⚡

صفحة هبوط تفاعلية متكاملة لـ **VOLT Recovery**، وهي منصة متخصصة في تقديم خدمات المساج المنزلي الفاخر والاستشفاء العضلي الرياضي في القاهرة الكبرى تحت إشراف "كابتن محمود". تم بناء وتطوير الصفحة باستخدام تقنيات الويب الأساسية مع التركيز على سرعة الأداء وسلاسة تجربة المستخدم (UX/UI).

---

## 📌 مميزات المشروع (Features)

* **تصميم عصري متجاوب (Responsive Design):** متوافق بالكامل مع مختلف الشاشات (أجهزة مكتبية، هواتف ذكية، أجهزة لوحية) مع واجهة مخصصة لكل بيئة عرض.
* **شريط تصفح سفلي ذكي للهواتف (Bottom App Navigation):** يمنح مستخدمي الهواتف تجربة شبيهة بتطبيقات الأجهزة المحمولة مع ميزة الـ Scrollspy لتحديد القسم النشط تلقائياً أثناء التمرير.
* **محرك جزيئات تفاعلي (HTML5 Canvas Particles):** خلفية ديناميكية خفيفة الوزن تعتمد على لغة JavaScript النقية دون الحاجة لمكتبات خارجية ثقيلة، مما يحافظ على سرعة تحميل الصفحة.
* **تأثيرات بصرية تفاعلية (Interactive UI):**
  * تأثير ثلاثي الأبعاد عند مرور الفأرة على بطاقات الأسعار (3D Tilt Effect).
  * تأثير تتبع مؤشر الفأرة (Cursor Glow) للأجهزة المكتبية.
  * عدادات تفاعلية متحركة عند الوصول إلى قسم الإحصاءات.
* **نظام إشعارات مدمج (Toast Notifications):** لعرض رسائل تفاعلية مرنة تزيد من تفاعل الزائر.
* **تهيئة متكاملة لمحركات البحث (SEO & Performance Optimized):**
  * مدمج بها بيانات مهيكلة (JSON-LD Structured Data) لتحسين ظهور الموقع في نتائج البحث المحلية.
  * استخدام وسوم الميتا (Meta Tags) المناسبة لمنصات التواصل الاجتماعي (Open Graph & Twitter Cards).

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **HTML5:** لبناء هيكل الصفحة بشكل متوافق مع معايير الويب القياسية.
* **CSS3:** لتنسيق الواجهة واستخدام المتغيرات (Custom Properties)، والـ Flexbox/Grid لتوزيع العناصر، بالإضافة إلى تقنيات الزجاج البلوري (Glassmorphism).
* **Vanilla JavaScript:** لتطوير كافة التأثيرات الحركية والتفاعلية دون الاعتماد على مكتبات خارجية، مما يضمن أداءً مستقراً.
* **FontAwesome:** لتوفير الأيقونات التفاعلية.
* **Google Fonts:** استخدام خطوط متناسقة تدعم اللغتين العربية والإنجليزية بشكل أنيق (Tajawal & Outfit).

## 🚀 طريقة التشغيل والاستخدام

يمكنك تشغيل المشروع محلياً وبكل سهولة باتباع الخطوات التالية:

1. **نسخ المستودع (Clone):**
   ```bash
   git clone https://github.com/YOUR_USERNAME/volt-recovery.git
   ```
2. **الانتقال إلى مجلد المشروع:**
   ```bash
   cd volt-recovery
   ```
3. **التشغيل:**
   * ببساطة قم بفتح ملف `index.html` مباشرة في أي متصفح ويب، أو استخدم إضافة مثل **Live Server** في بيئة VS Code لتجربة تطوير أفضل.
